# Student Management System (C++)

## 1. Introduction
This project is a simple Student Management System developed using C++. It helps manage student records.

## 2. Objectives
- To store student data
- To add, display and search students

## 3. Methodology / System Design
The system is based on Object-Oriented Programming.
A class "Student" is used with functions to input and display data.

## 4. Implementation
The project is implemented using C++.
It uses arrays and menu-driven program logic.

## 5. Conclusion
This project helped in understanding basic OOP concepts like classes and objects.
